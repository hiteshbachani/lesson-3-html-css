What is flex in CSS?

CSS is a magic, you all agree that makes our webpages beautiful and more presentable but do you know what is display: flex in CSS? If not after reading this you'll surely get the gist of it.

Whenever you're defining display to be flex i.e. display: flex, what you're saying to a computer is that "Hey you computer, I want flexible boxes". Computer says - Ok, done! you defined the display as flex I'll give you flexible box, but

you say - but what? computer says - but to make those boxes flexible I'll be needing something static on which they can flex. you say - oh okay, okay! here it is - I'm defining a container with its fixed width & height you define a container & inside that container you put flex boxes - say 1, 2 or how many you want.

for eg: you want image & text & then a button so this is how you can do:

div class = flex-container> => it defines the fixed container inside this follwing boxes will be flex

div {image} div => your image
div {text} div => your text
div {button} div => your button

this is basically the core concept of using flex.
